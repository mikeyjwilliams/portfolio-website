sweet Eats Project

Focus

In the Sweet Eats Project, the main focus is on the CSS stylesheet. This project was to help with learning about CSS specificity, weight, and inheritance. This was a more basic project, but still important to see the difference when you add specific classes or use a reset on your stylesheet and what it does to it.

Lessons

Playing with the styles, some of the simpler examples were the weight of an ID over a class but still important to realize. A large one to understand was how important it was pointed out to not use the style '!important' and its consequences when used. There is a lot of legacy code out there and not all of it is wrong, but not all of it is Best Practice by newer standards, this is something that must be kept in mind when finding examples and looking around websites in the console. Some sites might use things that are outdated, some might need to keep to certain standards, and some might just be on a certain budget. This does not mean we have to practice everything we see or should and must decipher what is correct form and what is incorrect.
